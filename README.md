# Amazon Homepage Clone

An Amazon-inspired homepage clone built using HTML and CSS.

This project was built as part of my frontend development journey to strengthen my understanding of HTML, CSS, layouts, Flexbox, and real-world UI design.

---

## ✨ Features

- Amazon-inspired homepage UI
- Navigation bar with logo, search bar, account, orders, and cart section
- Hero banner section
- Product category cards with images
- Multi-section footer
- Hover effects for interactive elements
- Clean and organized folder structure
- Built using semantic HTML and modern CSS

---

## 🛠️ Tech Stack

- HTML
- CSS
- Font Awesome
